# weather

a weather pwa application

[https://www.tjwyz.com](https://www.tjwyz.com)

## Usage

```
npm install

npm install -g  ppack

npm install -g  pm2

//make your pack.config.js first in your worker directory and then

npm run init

```

you can write pack.config.js like this

```
module.exports = {
	src:'./src',
	output:'./public'
}
```
